## 꼭 알아야할 ES6+ 문법

상수/변수 선언

var 대신에 const 혹은 let을 사용 -> block scope

* const : 재할당 불가. 내부 속성값은 수정 가능.
* let : Lexical Variable Scoping을 지원하는 변수 선언 문법

```javascript
console.log(name);  // undefined
console.log(lang);  // ReferenceError

var name = "Seolyu";

console.log(name);
```

실제로

```javascript
var name;  // 선언이 위로
console.log(name);

var name = "Seolyu";

console.log(name);
```

var 말고

const, let 을 사용하면 에러 발생

선언되고 나서야 그 변수 사용 가능



const 는 선언할 때 값을 넣어야하고 선언 후 값 바꾸는거 X

```javascript
let name;  // undefined

console.log(name);

name = "Seolyu";

console.log(name);
```



const라도 object 안 값 변경은 가능

```javascript
const tom = { lang: "Python" };
tom.lang = "Javascript";

console.log(tom);
```



var/let 사용여부에 따라 alert 메세지가 달라짐

```javascript
<body>
</body>

<script>
    var div;
    var container = document.getElementsByTagName('body')[0];
    for(let i=0; i<5; i++) {
        div = document.createElement('div');
        div.onclick = function() {
            alert("clicked : #" + i);
        };
        div.innerHTML = "#" +i;
        container.appendChild(div);
    }
</script>
```





## Object 선언

```javascript
let tom1 = {
    name: "Tom",
    age: 10,
    region: "Seoul"
};
```

```javascript
let tom2 = {
    "name": "Tom",
    "age": 10,
    "region": "Seoul"
};
```

위 두 코드는 동일



파이썬은

```python
tom = {
    "name": "Tom",
    "age": 10,
    "region": "Seoul"
}
```
